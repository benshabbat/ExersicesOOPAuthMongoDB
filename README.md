## ğŸ“– Core OOP Principles

<table>
<tr>
<td width="50%">

### 1ï¸âƒ£ Encapsulation
Hiding internal details and exposing only what's necessary:
```javascript
class BankAccount {
    #balance; // Private
    
    get balance() {
        return this.#balance;
    }
}
```

### 2ï¸âƒ£ Inheritance
Creating new classes from existing ones:
```javascript
class Animal {
    eat() { }
}

class Dog extends Animal {
    bark() { }
}
```

</td>
<td width="50%">

### 3ï¸âƒ£ Polymorphism
Different objects responding differently to the same method:
```javascript
class Shape {
    getArea() { return 0; }
}

class Circle extends Shape {
    getArea() { 
        return Math.PI * this.radius ** 2; 
    }
}
```

### 4ï¸âƒ£ Abstraction
Hiding complexity and exposing a simple interface:
```javascript
class Calculator {
    #complexCalculation() { }
    
    calculate(a, b) {
        return this.#complexCalculation();
    }
}
```

</td>
