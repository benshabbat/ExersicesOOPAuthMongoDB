# תרגיל 02 - בדיקות פונקציות 🔧

## מטרה
ללמוד לבדוק פונקציות - לקרוא להן עם פרמטרים ולבדוק את התוצאות.

## רקע
בתכנות אמיתי, אנחנו בודקים פונקציות שכתבנו. נלמד איך לבדוק שפונקציה מחזירה את הערך הנכון.

## תיאוריה

### מבנה בדיקת פונקציה

```javascript
// הפונקציה שאנחנו בודקים
function add(a, b) {
  return a + b;
}

// הבדיקה
test('פונקציית חיבור עובדת', () => {
  const result = add(2, 3);
  assert.strictEqual(result, 5);
});
```

### טיפים
- קרא לפונקציה עם ערכים ידועים
- שמור את התוצאה במשתנה
- בדוק שהתוצאה היא מה שציפית
