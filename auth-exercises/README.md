## ‚ùì Frequently Asked Questions

**Q: What is localhost:3000?**
A: It's your computer's address. 3000 is the port number.

**Q: How do I stop the server?**
A: Press Ctrl+C in the terminal.

**Q: Server not working, what should I do?**
A: 
1. Check that you installed packages (npm install)
2. Check for code errors
3. Check that port 3000 is not already in use

**Q: How do I know if my code is correct?**
A: Send a request to the server and verify the response matches expected output.
