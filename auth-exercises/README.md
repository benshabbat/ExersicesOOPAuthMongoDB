# Authentication Exercises - JWT, bcrypt & Express

This folder contains progressive exercises for learning authentication with:
- **Express.js** - Web framework for Node.js
- **bcrypt** - Password hashing library
- **JWT (JSON Web Tokens)** - Token-based authentication

## Prerequisites

Install the required packages:
```bash
npm init -y
npm install express bcrypt jsonwebtoken dotenv
npm install --save-dev nodemon
```

## Exercise Structure

Each exercise includes:
- `.md` file with instructions and requirements
- `.js` file for your implementation

## Exercises

1. **Exercise 01** - Express Server Setup
2. **Exercise 02** - Password Hashing with bcrypt
3. **Exercise 03** - JWT Token Creation and Verification
4. **Exercise 04** - User Registration Endpoint
5. **Exercise 05** - User Login Endpoint
6. **Exercise 06** - Protected Routes with Middleware
7. **Exercise 07** - Token Refresh System
8. **Exercise 08** - Complete Authentication System

## Testing

You can test your endpoints using:
- Postman
- Thunder Client (VS Code extension)
- curl commands
- REST Client (VS Code extension)

## Getting Started

Start with exercise-01 and work your way through each exercise sequentially. Each exercise builds upon the previous ones.
